Github: https://github.com/callumdmay/java-tower-defense/tree/master

Open project in Eclipse IDE by going to File -> Import -> General -> Projects Under file or archive. Use MacOS (Different lwjgl files required to run game with another OS)

Run Game.java to play game. Run test files in test folder to run tests.

For mock tests, add '-Dnet.bytebuddy.experimental=true' under VM arguments in Run Configurations.

index.html contains jacoco coverage report.
TowerDefense_Usability_Report.pdf contains usability report.
slides.pdf contains presentation slides.

Screenshots folder contains screenshots of tests running and coverage.
